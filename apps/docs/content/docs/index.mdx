This documentation app is the project hub for the current workspace. It is built with TanStack Start, TanStack Router, Fumadocs MDX, and Fumadocs search, but the content is focused on the code that actually exists in this repository.

<Callout type="info" title="What this site covers">
The main demo surface is the shared universal todo flow in `packages/todo-universal`, plus the shell apps and packages around it.
</Callout>

- A custom landing page at `/` that acts like the product/demo entrypoint.
- Search via `/api/search`.
- LLM-friendly text exports via `/llms.txt`, `/llms-full.txt`, and `/llms.mdx/docs/*`.
- A docs tree that explains the current repository shape instead of generic starter content.

## Core stack

| Layer | Role in this app |
| --- | --- |
| `@tanstack/react-start` | SSR-capable app shell and route runtime |
| `@tanstack/react-router` | File routes, loaders, server handlers, and navigation |
| `fumadocs-mdx` | Content pipeline for the docs tree |
| `fumadocs-ui` | Layouts, search UI, page tree, and doc page primitives |
| `lucide-react` | Icons used by the docs surface and MDX pages |
